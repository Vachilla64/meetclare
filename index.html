The Clari-fication

She's not magic.
She's just very good at SQL.

The Engine Room

Powered by Coral

Clare is built on Coral — the open-source SQL runtime for AI agents. Every data source is a SQL table. Every question is a query.

-- Clare asks: "What does my day look like?"

SELECT summary, status
  FROM google_calendar.events
  LIMIT 5;

SELECT number, title, state
  FROM github.issues
  WHERE owner='withcoral'
    AND state='open'
  LIMIT 5;

-- No ETL. No warehouse. Just answers.
